# Stremio FilmTV.it Addon

A Stremio addon that provides curated movie lists from FilmTV.it, including "Best of Year" collections.

## Installation in Stremio

To install this addon in Stremio, click the following link or copy it to your browser:

```
https://YOUR_HF_USERNAME-filmtv-x-stremio.hf.space/manifest.json
```

Or manually:
1. Open Stremio
2. Go to Addons
3. Click the puzzle icon (top right)
4. Paste the manifest URL above
5. Click "Install"

## Features

- Browse best movies by year (2023, 2024, 2025)
- Movie metadata including ratings, genres, directors, and cast
- Automatic caching to reduce server load
